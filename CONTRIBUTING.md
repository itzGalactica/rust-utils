# Contributions to rust_utils
Rust_utils exists purely as a dependancy for my other Rust projects.
Please note that *contributions for new functionality are currently not
being accepted.* However, if you find a bug in the already-existing
functionality, by all means open an issue.

The usual rules about issues apply:
- Ensure your issue isn't a duplicate before you submit it
- Try to add as much detail as possible, including steps to reproduce the bug
- Include as much relevant info as possible (e.g. version, error messages, etc.)

If you think you can fix the issue, open an PR and link your issue to it.
Usual code contributions apply:
- Keep your code style consistent with the project and Rust style guidelines
- Try to write idiomatic Rust code and avoid any bad programming practices and hacky implementations
- Comment code that might not be obvious, and keep comments clear but concise
- Don't add or change any functionality
